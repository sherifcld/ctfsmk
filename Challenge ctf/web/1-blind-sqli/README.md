# Challenge: Blind SQLi
**Category:** Web Exploitation
**Difficulty:** Medium

## Description
Sistem login kami sangat aman. Kami tidak akan memberitahu jika password salah, hanya apakah login berhasil atau tidak.

## Flag
`CTF{bl1nd_sql1_1s_t3d10us_but_w0rth_1t}`

## Solution
Gunakan serangan Blind SQL Injection (Boolean-based atau Time-based). Kamu bisa menggunakan `sqlmap` atau menulis script Python sendiri untuk mengekstrak password karakter demi karakter.
