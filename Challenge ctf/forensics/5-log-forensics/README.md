# Challenge: Log Forensics
**Category:** Forensics
**Difficulty:** Hard

## Description
Seorang penyusup berhasil masuk ke server kami. Cari tahu apa yang mereka ambil!

## Flag
`CTF{th3_tr41l_l34ds_t0_th3_r00t}`

## Solution
Periksa `auth.log` untuk melihat aktivitas login, `.bash_history` untuk perintah yang dijalankan, dan `config.php` untuk informasi tambahan yang mungkin ditinggalkan.
