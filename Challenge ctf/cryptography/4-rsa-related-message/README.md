# Challenge: RSA Related Message
**Category:** Cryptography
**Difficulty:** Ultimate Hard

## Description
Dua pesan yang sangat mirip ($m_2 = m_1 + 1337$) dienkripsi dengan modulus yang sama.

## Flag
`CTF{fr4nkl1n_r31t3r_r3l4t3d_m3ss4g3_4tt4ck}`

## Solution
Gunakan Franklin-Reiter Related Message Attack. Kamu bisa menggunakan resultan polinomial atau algoritma GCD polinomial untuk menemukan $m$.
