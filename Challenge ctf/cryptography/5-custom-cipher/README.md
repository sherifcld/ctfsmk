# Challenge: Hidden Number Problem
**Category:** Cryptography
**Difficulty:** Ultimate Hard

## Description
Kami memberikan beberapa bit paling signifikan dari $(a_i \cdot \alpha) \pmod{p}$. Bisakah kamu menemukan $\alpha$?

## Flag
`CTF{l4tt1c3_b4s3d_hnp_1s_r34lly_h4rd_992}`

## Solution
Ini adalah Hidden Number Problem yang bisa diselesaikan dengan mereduksi kisi (lattice reduction) menggunakan algoritma LLL atau BKZ.
