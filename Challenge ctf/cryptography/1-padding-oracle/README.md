# Challenge: Padding Oracle
**Category:** Cryptography
**Difficulty:** Medium

## Description
Tukang roti kami kehilangan kunci untuk membuka oven. Bisakah kamu membantunya? Server ini memungkinkan kamu untuk mendekripsi pesan, tetapi hanya memberitahu jika paddingnya valid.

## Flag
`CTF{p4dd1ng_0r4cl3_1s_st1ll_r3l3v4nt_2026}`

## Solution
Gunakan serangan Padding Oracle untuk mendekripsi ciphertext blok demi blok. Kamu bisa menggunakan tools seperti `padbuster` atau script Python kustom.
